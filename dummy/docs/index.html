▸ splitHours(site, hoursWorked, totalHoursThisWeek, consecutiveDayCount, correspondingPersonObject, correspondingDateObject)
admin/main-compiled.js

Splits a sum of hours into regular, overtime, and double-time hours according to state. Then separates the data. This is done for each day in a pay period.

splitHours(site: string, hoursWorked: number, totalHoursThisWeek: number, consecutiveDayCount: any, correspondingPersonObject: object, correspondingDateObject: object): object
Parameters
site (string) Site location. Either: "sw", "stan", "ucf", or "villa".
hoursWorked (number) Total number of hours worked.
totalHoursThisWeek (number) Total hours worked in the given work week.
consecutiveDayCount (any)
correspondingPersonObject (object) An object corresponding to the staff member that is being worked on.
correspondingDateObject (object) A date object corresponding to what time this is. *Unused by the function. Need to refactor this.
Returns
object: An object containing: regHours, otHours, dtHours, and sickHours.
Example

Split up the hours, if the staff member worked 11 hours in the day, 40 hours this week, has worked 5 days in a row, and worked at Villanova University.

splitHours("villa", 11, 40, 5, {correspondingPersonObject});
// Returns {regHours: 8, otHours: 3, dtHours: 0, sickHours: 0};

▸ checkIfRound(days, rate, rateUnrounded)
admin/main-compiled.js

This function is used for salary employees. It takes in the days, and 2 rates - a rounded and an unrounded rate. The function returns a dailyPayRate depending on the number of days.

checkIfRound(days: number, rate: number, rateUnrounded: number): number
Parameters
days (number) The amount of days the employee worked.
rate (number) The pay rate for the employee. (ex. $950 / week)
rateUnrounded (number) The unrounded rate.
Returns
number: The daily rate for the salary employee, rounded correctly so as not to deviate from the correct amount. The returned value will later be used by moneyString().
Example

Check the rate for a 1st year FAC working 1 day.

checkIfRound(1, 154.17, 154.166666667);
// returns 154.17

Check the rate for a 1st year FAC working 5 days.

checkIfRound(5, 154.17, 154.166666667);
// returns 770.8333333333;
